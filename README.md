# 🚀 Portal de Estudos - Prof(a) Thalia Ellen

Landing page desenvolvida como porta de entrada para conteúdos educacionais de **HTML** e **Programação Orientada a Objetos (P.O.O.)**, utilizada em sala de aula com alunos do curso de Informática do Frei Orlando.

## 🎯 Objetivo

Durante minha atuação como professora bolsista, identifiquei a necessidade de criar **materiais digitais rápidos, acessíveis e organizados**, permitindo que os alunos revisassem conteúdos de forma prática fora da sala de aula.

Este projeto surgiu como uma solução para:

* Centralizar conteúdos de estudo
* Facilitar o acesso dos alunos
* Criar uma experiência moderna e intuitiva

## 🤖 Uso de Inteligência Artificial

Aproveitei o contexto das aulas para estudar e aplicar o uso de **IA na geração de interfaces**, focando em:

* Criação rápida de layouts funcionais
* Prototipação de interfaces educacionais
* Otimização do tempo de desenvolvimento

💡 Em um cenário tradicional, essa interface demandaria um tempo significativamente maior para ser construída do zero.

Aqui, a IA foi utilizada como **ferramenta de aceleração**, enquanto o refinamento, estrutura e adaptação ao contexto educacional foram conduzidos por mim.

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* Design responsivo
* Conceitos de UI/UX
* Apoio de Inteligência Artificial (ChatGPT)

## 🧪 Aplicação em sala de aula

O portal foi utilizado como:

* Ambiente de apoio para estudos
* Base para navegação entre conteúdos
* Exemplo prático de aplicação de tecnologia no ensino

Além disso, também serviu como ponto de discussão com os alunos sobre:

* Uso consciente de IA
* Produtividade no desenvolvimento
* Boas práticas de organização de projetos

Este projeto reflete minha abordagem de:

> unir tecnologia, ensino e inovação de forma prática.
