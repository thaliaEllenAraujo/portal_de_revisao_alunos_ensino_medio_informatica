Playlist de Música

Vetor Função
🎸
Imagine que você tem uma prateleira com 10 CDs, cada um em um lugar numerado: o CD número 1 fica na posição 1, o CD 2 na posição 2, e por aí vai. No Java, essa prateleira se chama vetor (ou array)!
👆
O usuário vai falar um número — tipo "quero ver o CD da posição 3" — e o programa vai até aquela posição e mostra o nome da música. Isso vai dentro de uma função, que é como um assistente especializado que faz uma tarefa específica.
⚠️
Atenção importante: no Java, as posições do vetor começam em 0, não em 1! Então quando o usuário digitar "3", você precisa buscar a posição 2 no vetor. É só fazer: posição do vetor = número do usuário - 1.
1
Criar a "prateleira" de músicas

Criamos um vetor do tipo String com 10 posições e já colocamos o nome de 10 músicas dentro. É como preencher a prateleira com os CDs antes de abrir a loja.

2
Mostrar todas as músicas na tela

Usamos um laço for para percorrer o vetor do início ao fim e mostrar cada música com seu número. O laço é como um robozinho que visita cada posição da prateleira.

3
Perguntar ao usuário qual ele quer ver

Pedimos para o usuário digitar um número de 1 a 10. Usamos o Scanner para capturar essa resposta — o Scanner é como um microfone que recebe o que o usuário digita.

4
Criar a função buscarMusica()

Essa função recebe a prateleira (vetor) e o número escolhido. Ela subtrai 1 do número para achar a posição certa e devolve o nome da música. É como um funcionário da loja que vai buscar o CD para você!

5
Verificar se o número é válido e mostrar o resultado

Antes de buscar, checamos se o número está entre 1 e 10. Se estiver, chamamos a função e mostramos a música. Se não estiver, avisamos que é inválido. Nunca confie que o usuário vai digitar certo! 😄

Playlist.java 
import java.util.Scanner;

public class Playlist {

    // Função que busca a música na posição indicada pelo usuário
    static String buscarMusica(String[] playlist, int posicao) {
        // O usuário digita de 1 a 10, mas o array começa em 0
        // Então subtraímos 1: posição 1 do usuário = índice 0 do array
        return playlist[posicao - 1];
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Criando o vetor com 10 músicas já preenchidas
        String[] playlist = {
            "Bohemian Rhapsody", "Imagine", "Hotel California",
            "Shape of You", "Blinding Lights", "Smells Like Teen Spirit",
            "Rolling in the Deep", "Uptown Funk", "Bad Guy", "Thriller"
        };

        // Mostrando todas as músicas numeradas (i+1 porque o índice começa em 0)
        System.out.println("=== PLAYLIST ===");
        for (int i = 0; i < playlist.length; i++) {
            System.out.println((i + 1) + ". " + playlist[i]);
        }

        // Pedindo ao usuário que escolha uma posição
        System.out.print("\nDigite o número da música (1 a 10): ");
        int numero = sc.nextInt();

        // Verificando se o número é válido antes de buscar
        if (numero >= 1 && numero <= 10) {
            String musica = buscarMusica(playlist, numero);
            System.out.println("Música escolhida: " + musica);
        } else {
            System.out.println("Número inválido!");
        }

        sc.close();
    }
}
📥 Entrada do usuário
Digite o número da música (1 a 10): 3
📤 Saída do programa
Música escolhida: Hotel California

Notas da Turma

Matriz Função
📋
Imagine uma folha de papel quadriculada: cada linha representa um aluno diferente, e cada coluna representa uma prova diferente. Isso é exatamente o que chamamos de matriz no Java — um vetor dentro de outro vetor!
🧮
Para calcular a média de cada aluno, pegamos todas as notas da linha daquele aluno, somamos e dividimos pelo número de provas. É igual ao que você faz na mão: some tudo e divida pela quantidade!
🔄
Vamos usar dois laços for um dentro do outro: o de fora passeia pelas linhas (pelos alunos) e o de dentro passeia pelas colunas (pelas provas). É como uma varredura — linha por linha, coluna por coluna.
1
Criar a "tabela" de notas (matriz)

Declaramos double[5][3] — isso cria 5 linhas (alunos) e 3 colunas (provas). É como desenhar uma tabela em branco no papel antes de preencher.

2
Preencher a tabela com dois laços for

O laço de fora conta os alunos (0, 1, 2, 3, 4). Para cada aluno, o laço de dentro pede as notas das 3 provas. É como preencher a tabela célula por célula.

3
Criar a função calcularMedia()

Essa função recebe um vetor de notas (uma linha da tabela) e retorna a média. Ela soma todos os valores e divide pelo tamanho do vetor. Reutilizamos ela para cada aluno!

4
Chamar a função e mostrar cada média

Para cada aluno, passamos a linha dele para a função e mostramos o resultado. notas[i] é a linha inteira do aluno i — o Java entende isso automaticamente!

NotasTurma.java 
import java.util.Scanner;

public class NotasTurma {

    // Função que calcula a média de um vetor de notas (uma linha da matriz)
    static double calcularMedia(double[] notas) {
        double soma = 0;
        // Percorre cada nota e vai acumulando na variável soma
        for (int i = 0; i < notas.length; i++) {
            soma += notas[i]; // soma += x  é igual a  soma = soma + x
        }
        // Divide a soma pela quantidade de provas para ter a média
        return soma / notas.length;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int alunos = 5; // número de alunos (linhas)
        int provas  = 3; // número de provas (colunas)

        // Criando a matriz: 5 linhas (alunos) x 3 colunas (provas)
        double[][] notas = new double[alunos][provas];

        // Dois laços: o externo para alunos, o interno para as provas
        for (int i = 0; i < alunos; i++) {
            System.out.println("--- Aluno " + (i + 1) + " ---");
            for (int j = 0; j < provas; j++) {
                System.out.print("  Nota da prova " + (j + 1) + ": ");
                notas[i][j] = sc.nextDouble(); // [linha][coluna]
            }
        }

        // Calculando e exibindo a média de cada aluno
        System.out.println("\n=== MÉDIAS ===");
        for (int i = 0; i < alunos; i++) {
            // notas[i] passa a linha inteira do aluno i para a função
            double media = calcularMedia(notas[i]);
            System.out.printf("Aluno %d: %.1f%n", (i + 1), media);
        }

        sc.close();
    }
}
📥 Entrada
Aluno 1: 7.0 / 8.0 / 9.0
Aluno 2: 5.0 / 6.0 / 7.0
📤 Saída
Aluno 1: 8.0
Aluno 2: 6.0

Jogo da Velha

MatrizFunção
⬛
O tabuleiro do jogo da velha tem 3 linhas e 3 colunas — uma matriz 3x3 perfeita! Cada casinha pode ter 'X', 'O', ou estar vazia. No começo, todas estão vazias.
🔄
O jogo alterna entre dois jogadores. Usamos um truque matemático simples: se o número da jogada for par → é o jogador X. Se for ímpar → é o jogador O. Isso funciona porque ímpar/par alterna sempre!
🏆
Para verificar se alguém ganhou, precisamos checar 8 combinações: 3 linhas + 3 colunas + 2 diagonais. Se todos os 3 valores de qualquer combinação forem iguais, esse jogador ganhou!
1
Criar o tabuleiro vazio

Criamos uma matriz char[3][3] e preenchemos tudo com espaço (' '). O espaço representa uma casinha vazia — ainda não foi jogada.

2
Criar a função mostrarTabuleiro()

Essa função percorre a matriz e desenha o tabuleiro com as linhas divisórias (---|---|---). É o "desenhista" do jogo — ele mostra como o tabuleiro está a cada rodada.

3
Criar a função verificarVencedor()

Essa função checa as 3 linhas, 3 colunas e 2 diagonais. Se os três valores de qualquer combinação forem do mesmo jogador, ela retorna true (verdadeiro) — esse jogador ganhou!

4
Criar o laço principal do jogo

Em um laço, alternamos entre X e O (usando o truque do par/ímpar), pedimos linha e coluna, colocamos a peça no tabuleiro, e verificamos se houve vencedor. O jogo termina com vitória ou após 9 jogadas (empate).

JogoDaVelha.java 
import java.util.Scanner;

public class JogoDaVelha {

    // Função que exibe o tabuleiro formatado com linhas divisórias
    static void mostrarTabuleiro(char[][] tab) {
        System.out.println("\n  1   2   3");
        for (int i = 0; i < 3; i++) {
            System.out.print((i + 1) + " ");
            for (int j = 0; j < 3; j++) {
                System.out.print(" " + tab[i][j] + " ");
                if (j < 2) System.out.print("|"); // divisória vertical
            }
            System.out.println();
            if (i < 2) System.out.println("  ---|---|---"); // divisória horizontal
        }
        System.out.println();
    }

    // Função que verifica se um jogador venceu (checa 8 combinações)
    static boolean verificarVencedor(char[][] tab, char jogador) {
        // Checa as 3 linhas horizontais
        for (int i = 0; i < 3; i++)
            if (tab[i][0] == jogador && tab[i][1] == jogador && tab[i][2] == jogador)
                return true;
        // Checa as 3 colunas verticais
        for (int j = 0; j < 3; j++)
            if (tab[0][j] == jogador && tab[1][j] == jogador && tab[2][j] == jogador)
                return true;
        // Checa a diagonal principal (↘) e a diagonal secundária (↙)
        if (tab[0][0] == jogador && tab[1][1] == jogador && tab[2][2] == jogador)
            return true;
        if (tab[0][2] == jogador && tab[1][1] == jogador && tab[2][0] == jogador)
            return true;
        return false; // ninguém ganhou ainda
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        char[][] tab = new char[3][3];

        // Preenche o tabuleiro com espaços (posições vazias)
        for (int i = 0; i < 3; i++)
            for (int j = 0; j < 3; j++)
                tab[i][j] = ' ';

        char[] jogadores = {'X', 'O'};
        int jogadas = 0;
        boolean fimDeJogo = false;

        while (!fimDeJogo && jogadas < 9) {
            // jogadas % 2 retorna 0 (par) ou 1 (ímpar) → alterna X e O
            char atual = jogadores[jogadas % 2];
            mostrarTabuleiro(tab);
            System.out.println("Jogador " + atual + ", é sua vez!");
            System.out.print("Linha (1-3): ");  int linha = sc.nextInt() - 1;
            System.out.print("Coluna (1-3): "); int col  = sc.nextInt() - 1;

            if (tab[linha][col] == ' ') { // verifica se a casinha está vazia
                tab[linha][col] = atual;
                jogadas++;
                if (verificarVencedor(tab, atual)) {
                    mostrarTabuleiro(tab);
                    System.out.println("Jogador " + atual + " VENCEU! 🎉");
                    fimDeJogo = true;
                }
            } else {
                System.out.println("Posição ocupada! Tente novamente.");
            }
        }
        if (!fimDeJogo) System.out.println("EMPATE!");
        sc.close();
    }
}
📥 Jogada
X: linha 1, col 1
O: linha 2, col 2
X: linha 1, col 2
...
📤 Resultado
X | X | X
---|---|---
| O |
Jogador X VENCEU! 🎉

Ranking de Pontuação

VetorFunção
🥇
Pense numa competição entre 8 jogadores. Para achar quem tem mais pontos, você olharia para o primeiro, depois para o segundo — se o segundo tiver mais, ele passa a ser o "campeão provisório". Você continua até o final. É exatamente isso que o computador vai fazer!
📍
A função vai guardar o índice (a posição) do maior valor, não o valor em si. Assim, no final sabemos tanto a posição quanto o valor: pontuacoes[indiceMaior].
1
Criar o vetor de pontuações

Declaramos int[8] para guardar as pontuações dos 8 jogadores. Então usamos um laço para perguntar a pontuação de cada um.

2
Criar a função encontrarMaior()

Começamos assumindo que o jogador 0 é o maior (índice 0). Aí percorremos do índice 1 até o 7. Se encontrarmos alguém com mais pontos, atualizamos quem é o campeão. No final, retornamos o índice do vencedor.

3
Mostrar o ranking e o vencedor

Exibimos todos os jogadores com suas pontuações e depois anunciamos o vencedor usando o índice retornado pela função. Lembrando que índice + 1 = número do jogador para o usuário!

Ranking.java 
import java.util.Scanner;

public class Ranking {

    // Função que retorna o ÍNDICE do jogador com maior pontuação
    static int encontrarMaior(int[] pontos) {
        int indiceMaior = 0; // começa assumindo que o primeiro é o maior
        for (int i = 1; i < pontos.length; i++) {
            // Se encontrar alguém com mais pontos, esse passa a ser o "líder"
            if (pontos[i] > pontos[indiceMaior]) {
                indiceMaior = i;
            }
        }
        return indiceMaior; // retorna a posição do vencedor
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int[] pontuacoes = new int[8];

        // Lendo as pontuações de 8 jogadores
        for (int i = 0; i < 8; i++) {
            System.out.print("Pontuação do jogador " + (i + 1) + ": ");
            pontuacoes[i] = sc.nextInt();
        }

        // Chamando a função para descobrir quem ganhou
        int idx = encontrarMaior(pontuacoes);

        // Exibindo o ranking completo
        System.out.println("\n=== RANKING ===");
        for (int i = 0; i < pontuacoes.length; i++) {
            System.out.println("Jogador " + (i + 1) + ": " + pontuacoes[i] + " pts");
        }

        // Anunciando o vencedor (idx+1 porque o índice começa em 0)
        System.out.println("\n🏆 Vencedor: Jogador " + (idx + 1) +
            " com " + pontuacoes[idx] + " pontos!");

        sc.close();
    }
}

390, 100, 275, 440
📤 Saída
🏆 Vencedor: Jogador 4
com 480 pontos!

Agenda Semanal

MatrizFunção
📆
Uma agenda tem 7 dias (linhas) e 3 horários por dia — manhã, tarde e noite (colunas). Isso forma uma matriz 7x3. Cada célula guarda o nome de um compromisso, ou "Livre" se não tem nada.
✏️
O usuário vai escolher o dia e o horário e digitar o compromisso. Isso vai atualizar a célula certa da matriz. É como riscar uma anotação numa agenda de papel!
1
Criar a matriz e preencher com "Livre"

Criamos String[7][3] e usamos dois laços para colocar "Livre" em todas as posições. É como comprar uma agenda nova — todas as páginas estão em branco!

2
Criar a função exibirAgenda()

Essa função mostra a agenda como uma tabela organizada, com os nomes dos dias e horários. Usamos printf para alinhar direitinho as colunas.

3
Criar a função marcarCompromisso()

Recebe o dia, o horário e o nome do evento, e atualiza a célula certa da matriz. Simples assim: agenda[dia][hora] = evento.

4
Criar um menu de opções

Usamos um laço do-while para mostrar as opções (marcar compromisso, ver agenda, sair). O programa continua rodando até o usuário escolher sair.

AgendaSemanal.java 
import java.util.Scanner;

public class AgendaSemanal {

    // Vetores com os nomes dos dias e horários (para facilitar a exibição)
    static String[] dias     = {"Seg","Ter","Qua","Qui","Sex","Sab","Dom"};
    static String[] horarios = {"Manhã","Tarde","Noite"};

    // Função que exibe a agenda como uma tabela organizada
    static void exibirAgenda(String[][] agenda) {
        System.out.println("\n=== AGENDA SEMANAL ===");
        System.out.printf("%-5s | %-15s | %-15s | %-15s%n",
            "Dia", "Manhã", "Tarde", "Noite");
        System.out.println("-".repeat(58));
        for (int i = 0; i < 7; i++) {
            System.out.printf("%-5s | %-15s | %-15s | %-15s%n",
                dias[i], agenda[i][0], agenda[i][1], agenda[i][2]);
        }
    }

    // Função que marca um compromisso em um dia e horário específicos
    static void marcarCompromisso(String[][] agenda, int dia, int hora, String evento) {
        agenda[dia][hora] = evento; // atualiza a célula da matriz
        System.out.println("✅ Compromisso marcado com sucesso!");
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String[][] agenda = new String[7][3];

        // Inicializa todas as posições com "Livre"
        for (int i = 0; i < 7; i++)
            for (int j = 0; j < 3; j++)
                agenda[i][j] = "Livre";

        int opcao;
        do {
            System.out.println("\n1-Marcar  2-Ver agenda  0-Sair");
            System.out.print("Opção: ");
            opcao = sc.nextInt();

            if (opcao == 1) {
                System.out.print("Dia (1=Seg a 7=Dom): ");
                int dia = sc.nextInt() - 1; // -1 para virar índice
                System.out.print("Horário (1=Manhã, 2=Tarde, 3=Noite): ");
                int hora = sc.nextInt() - 1;
                sc.nextLine(); // limpa o buffer do teclado
                System.out.print("Evento: ");
                String evento = sc.nextLine();
                marcarCompromisso(agenda, dia, hora, evento);
            } else if (opcao == 2) {
                exibirAgenda(agenda);
            }
        } while (opcao != 0);

        sc.close();
    }
}

Evento: Reunião
📤 Saída

✅ Compromisso marcado!

Temperaturas da Semana

VetorFunção
☀️
Temos 7 temperaturas, uma por dia — um vetor de 7 posições! Para calcular a média, é como você faria na escola: some todas e divida por 7.
🔥❄️
Além da média, podemos achar o dia mais quente e o mais frio da semana usando a mesma lógica do exercício anterior: percorrer o vetor guardando o índice do maior e do menor valor.
1
Criar um vetor de 7 temperaturas

Declaramos double[7] e lemos a temperatura de cada dia. Usamos double porque temperatura pode ter vírgula (ex: 28,5°C).

2
Criar a função calcularMedia()

Usamos o for-each para somar todos os valores. O for-each é uma versão simplificada do for que percorre todos os elementos automaticamente, sem precisar de índice.

3
Criar funções para o maior e menor

Seguindo a mesma lógica do ranking: começa pelo índice 0 e substitui sempre que encontrar um valor maior (ou menor). No final retorna o índice do dia mais quente (ou mais frio).

4
Mostrar os resultados formatados

Usamos printf com %.1f para mostrar apenas 1 casa decimal. O vetor dias[] nos ajuda a mostrar o nome do dia ao invés do número.

Temperaturas.java 
import java.util.Scanner;

public class Temperaturas {

    static String[] dias = {"Segunda","Terça","Quarta","Quinta","Sexta","Sábado","Domingo"};

    // Calcula a média usando for-each (mais simples que o for normal)
    static double calcularMedia(double[] temps) {
        double soma = 0;
        for (double t : temps) soma += t; // "para cada t em temps, soma t"
        return soma / temps.length;
    }

    // Retorna o índice do dia com maior temperatura
    static int diaMaisQuente(double[] temps) {
        int idx = 0;
        for (int i = 1; i < temps.length; i++)
            if (temps[i] > temps[idx]) idx = i;
        return idx;
    }

    // Retorna o índice do dia com menor temperatura
    static int diaMaisFrio(double[] temps) {
        int idx = 0;
        for (int i = 1; i < temps.length; i++)
            if (temps[i] < temps[idx]) idx = i; // diferença: aqui é <
        return idx;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        double[] temps = new double[7];

        for (int i = 0; i < 7; i++) {
            System.out.print("Temperatura de " + dias[i] + ": ");
            temps[i] = sc.nextDouble();
        }

        double media  = calcularMedia(temps);
        int    quente = diaMaisQuente(temps);
        int    frio   = diaMaisFrio(temps);

        System.out.printf("%nMédia da semana:  %.1f°C%n", media);
        System.out.printf("Dia mais quente:  %s (%.1f°C)%n", dias[quente], temps[quente]);
        System.out.printf("Dia mais frio:    %s (%.1f°C)%n", dias[frio],   temps[frio]);

        sc.close();
    }
}

31.0, 26.5, 29.0
📤 Saída

Mais quente: Quinta (32.5°C)
Mais frio: Sábado (26.5°C)

Loja de Produtos

MatrizFunção
🗂️
Cada produto tem 3 informações: nome, preço e quantidade. Com 5 produtos, temos 5 linhas e 3 colunas — uma matriz! Como nome é texto e preço/quantidade são números, vamos guardar tudo como String e converter quando precisar calcular.
💰
O valor total de um produto no estoque é simples: preço × quantidade. Para saber o total da loja inteira, somamos o valor de cada produto. A função vai percorrer a matriz, fazer a conta de cada linha e acumular o total.
1
Criar a matriz de produtos

Usamos String[5][3] onde cada linha tem: [nome, preço, quantidade]. Preenchemos diretamente no código — não precisa pedir para o usuário desta vez.

2
Criar a função calcularTotalEstoque()

Para cada produto (linha), convertemos o preço (String → double) e a quantidade (String → int) usando Double.parseDouble() e Integer.parseInt(). Multiplicamos e acumulamos.

3
Exibir a tabela e o total

Mostramos cada produto com seu subtotal (preço × qtd) e no final mostramos o total geral calculado pela função.

LojaProdutos.java 
public class LojaProdutos {

    // Função que calcula o valor total de todo o estoque
    static double calcularTotalEstoque(String[][] produtos) {
        double total = 0;
        for (int i = 0; i < produtos.length; i++) {
            // Converte String para número para poder multiplicar
            double preco = Double.parseDouble(produtos[i][1]);  // coluna 1 = preço
            int    qtd   = Integer.parseInt(produtos[i][2]);   // coluna 2 = quantidade
            total += preco * qtd; // acumula o subtotal de cada produto
        }
        return total;
    }

    public static void main(String[] args) {
        // Matriz: cada linha é [nome, preço (String), quantidade (String)]
        String[][] produtos = {
            {"Teclado",  "150.00", "10"},
            {"Mouse",    "80.00",  "25"},
            {"Monitor",  "900.00", "5" },
            {"Headset",  "200.00", "15"},
            {"Webcam",   "250.00", "8" }
        };

        // Exibindo a tabela formatada
        System.out.println("=== ESTOQUE DA LOJA ===");
        System.out.printf("%-10s | %-10s | %-5s | %-12s%n",
            "Produto", "Preço", "Qtd", "Subtotal");
        System.out.println("-".repeat(46));

        for (String[] p : produtos) { // for-each: p representa cada produto
            double preco = Double.parseDouble(p[1]);
            int    qtd   = Integer.parseInt(p[2]);
            System.out.printf("%-10s | R$%-8.2f | %-5d | R$%-10.2f%n",
                p[0], preco, qtd, preco * qtd);
        }

        System.out.println("-".repeat(46));
        System.out.printf("TOTAL DO ESTOQUE: R$%.2f%n",
            calcularTotalEstoque(produtos));
    }
}

Inverter Lista

VetorFunção
📚
Imagine uma pilha de 5 livros numerados de 1 a 5. Para mostrar em ordem inversa, você pega o de cima primeiro (o 5), depois o 4, depois o 3... É isso que o computador vai fazer: percorrer o vetor de trás para frente!
🔢
Se o vetor tem 5 posições, os índices são 0, 1, 2, 3, 4. Para inverter, começamos no índice 4 (o último) e vamos diminuindo até chegar no 0. O truque do índice final é: tamanho - 1.
1
Criar e preencher o vetor

Declaramos int[5] e pedimos 5 números ao usuário em um laço. Mostramos também a ordem original para comparar depois.

2
Criar a função imprimirInvertido()

O segredo está no laço for: em vez de começar em 0 e ir crescendo, começamos em numeros.length - 1 (o último índice) e vamos diminuindo até chegar em 0.

3
Chamar a função e mostrar o resultado

Chamamos a função e mostramos os números separados por vírgula. A lógica é igual ao original, mas agora o i começa no final e vai diminuindo: i-- ao invés de i++.

InverterLista.java 
import java.util.Scanner;

public class InverterLista {

    // Função que imprime os elementos do vetor ao contrário
    static void imprimirInvertido(int[] numeros) {
        System.out.print("Invertido: ");
        // Começa no último índice (length-1) e vai diminuindo até 0
        for (int i = numeros.length - 1; i >= 0; i--) {
            System.out.print(numeros[i]);
            if (i > 0) System.out.print(", "); // vírgula entre os números
        }
        System.out.println();
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int[] numeros = new int[5];

        System.out.println("Digite 5 números:");
        for (int i = 0; i < 5; i++) {
            System.out.print("Número " + (i + 1) + ": ");
            numeros[i] = sc.nextInt();
        }

        // Mostrando na ordem original
        System.out.print("Original:  ");
        for (int i = 0; i < numeros.length; i++) {
            System.out.print(numeros[i]);
            if (i < numeros.length - 1) System.out.print(", ");
        }
        System.out.println();

        // Chamando a função que imprime ao contrário
        imprimirInvertido(numeros);

        sc.close();
    }
}

10, 20, 30, 40, 50
📤 Saída
Original: 10, 20, 30, 40, 50
Invertido: 50, 40, 30, 20, 10

Mapa do Tesouro

MatrizFunção
🏝️
Imagine um mapa quadriculado 5x5 — exatamente uma matriz! O computador vai esconder o tesouro num lugar aleatório (usando aleatoriedade do Java) e o usuário vai tentando adivinhar a posição.
🎲
Math.random() gera um número entre 0 e 1 (como 0,7342...). Para transformar em um número inteiro de 0 a 4, multiplicamos por 5 e convertemos para inteiro: (int)(Math.random() * 5). Simples!
🙈
O mapa mostramos com ~ (til) em todas as posições — mesmo onde está o tesouro! Só verificamos internamente se o usuário acertou. É como um jogo de campo minado.
1
Criar o mapa 5x5 preenchido com '~'

Declaramos char[5][5] e preenchemos tudo com til (~), que representa a água/terreno desconhecido.

2
Sortear a posição do tesouro

Usamos Math.random() para sortear linha e coluna entre 0 e 4. Colocamos 'T' nessa posição da matriz — mas o usuário não vai ver!

3
Criar a função mostrarMapa()

Percorre a matriz e mostra '~' em todo lugar — inclusive onde está o 'T'. Isso esconde o tesouro. Quando o usuário erra, marcamos a posição com 'X'.

4
Dar 3 tentativas ao usuário

Em um laço de 3 rodadas, pedimos linha e coluna. Se acertar, fim de jogo! Se errar, marcamos 'X' naquele lugar e mostramos o mapa atualizado.

MapaTesouro.java 
import java.util.Scanner;

public class MapaTesouro {

    // Mostra o mapa SEM revelar onde está o tesouro
    static void mostrarMapa(char[][] mapa) {
        System.out.println("\n    1  2  3  4  5");
        for (int i = 0; i < 5; i++) {
            System.out.print((i + 1) + "  ");
            for (int j = 0; j < 5; j++) {
                // Se for 'T' (tesouro), mostra '~' para esconder
                char c = (mapa[i][j] == 'T') ? '~' : mapa[i][j];
                System.out.print(" " + c + " ");
            }
            System.out.println();
        }
    }

    // Verifica se a posição digitada é onde está o tesouro
    static boolean verificarPosicao(char[][] mapa, int l, int c) {
        return mapa[l][c] == 'T'; // true = acertou!
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        char[][] mapa = new char[5][5];

        // Preenche o mapa com água (~)
        for (int i = 0; i < 5; i++)
            for (int j = 0; j < 5; j++)
                mapa[i][j] = '~';

        // Sorteia a posição do tesouro (0 a 4 para linha e coluna)
        int tesL = (int)(Math.random() * 5);
        int tesC = (int)(Math.random() * 5);
        mapa[tesL][tesC] = 'T'; // esconde o tesouro aqui

        System.out.println("🗺️ Você tem 3 tentativas para achar o tesouro!");
        boolean achou = false;

        for (int t = 1; t <= 3 && !achou; t++) {
            mostrarMapa(mapa);
            System.out.println("Tentativa " + t + " de 3");
            System.out.print("Linha (1-5): ");   int l = sc.nextInt() - 1;
            System.out.print("Coluna (1-5): ");  int c = sc.nextInt() - 1;

            if (verificarPosicao(mapa, l, c)) {
                achou = true;
                System.out.println("🎉 PARABÉNS! Você achou o tesouro!");
            } else {
                mapa[l][c] = 'X'; // marca onde o usuário tentou
                System.out.println("❌ Errou! Tente novamente.");
            }
        }

        if (!achou)
            System.out.println("Fim de jogo! O tesouro estava em ("
                + (tesL+1) + "," + (tesC+1) + ")");

        sc.close();
    }
}

Linha: 3, Coluna: 4
📤 Resultado
🎉 PARABÉNS! Você achou o tesouro!
(ou: ❌ Errou! — posição aleatória)

Soma de Linhas

MatrizFunção
🔢
Uma matriz 3x3 tem 3 linhas. Para somar cada linha separadamente, precisamos somar só os elementos daquela linha — não misturar com as outras. É como somar os itens de cada prateleira independentemente.
🔁
O laço externo visita cada linha (0, 1, 2). Para cada linha, reiniciamos o contador de soma do zero. O laço interno soma os 3 valores daquela linha. Quando termina, mostramos a soma antes de passar para a próxima linha.
1
Criar e preencher a matriz 3x3

Declaramos int[3][3] e usamos dois laços aninhados para pedir os 9 valores ao usuário, posição por posição.

2
Criar a função somarLinha()

Recebe um vetor (uma linha da matriz) e retorna a soma de todos seus elementos com o for-each. Simples e reutilizável!

3
Mostrar a matriz com as somas

Para cada linha, mostramos os valores entre colchetes e chamamos a função para calcular a soma. matriz[i] passa a linha i inteira para a função — o Java entende que uma linha da matriz é um vetor!

SomaLinhas.java 
import java.util.Scanner;

public class SomaLinhas {

    // Função que soma todos os elementos de um vetor (uma linha da matriz)
    static int somarLinha(int[] linha) {
        int soma = 0;
        // for-each: "para cada valor na linha, adicione à soma"
        for (int valor : linha) {
            soma += valor;
        }
        return soma;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int[][] matriz = new int[3][3];

        // Dois laços para preencher a matriz posição por posição
        System.out.println("Digite os valores da matriz 3x3:");
        for (int i = 0; i < 3; i++) {
            System.out.println("--- Linha " + (i + 1) + " ---");
            for (int j = 0; j < 3; j++) {
                System.out.print("  [" + (i+1) + "][" + (j+1) + "]= ");
                matriz[i][j] = sc.nextInt();
            }
        }

        // Exibindo a matriz e a soma de cada linha
        System.out.println("\n=== MATRIZ E SOMAS ===");
        for (int i = 0; i < 3; i++) {
            System.out.print("Linha " + (i + 1) + ": [ ");
            for (int j = 0; j < 3; j++) {
                System.out.print(matriz[i][j] + " ");
            }
            // matriz[i] passa a linha inteira como vetor para a função
            System.out.println("] → Soma = " + somarLinha(matriz[i]));
        }

        sc.close();
    }
}

Linha 1: 1, 2, 3
Linha 2: 4, 5, 6
Linha 3: 7, 8, 9
📤 Saída
Linha 1: [ 1 2 3 ] → Soma = 6
Linha 2: [ 4 5 6 ] → Soma = 15
Linha 3: [ 7 8 9 ] → Soma = 24
