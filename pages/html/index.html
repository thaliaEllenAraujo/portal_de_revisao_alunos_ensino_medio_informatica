01

Estrutura básica de um arquivo HTML

Todo site começa com essa estrutura. Crie um arquivo chamado index.html, abra no bloco de notas ou VS Code, e cole o código abaixo. Depois abra no navegador!
HTML 
<!-- index.html — todo site começa assim -->
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Meu Primeiro Site</title>
  <link rel="stylesheet" href="estilo.css">
</head>
<body>

  <!-- Tudo que aparece na tela vai aqui -->
  <h1>Olá, mundo!</h1>

</body>
</html>
💡 O arquivo index.html é sempre a página principal. O arquivo estilo.css fica na mesma pasta e cuida das cores e do visual.
02

Textos e títulos

HTML tem 6 tamanhos de título (h1 a h6) e uma tag para parágrafos. O CSS controla a aparência deles.
HTML 
<h1>Título principal (maior)</h1>
<h2>Título secundário</h2>
<h3>Subtítulo</h3>
<p>Este é um parágrafo de texto normal.</p>
<p>Você pode deixar texto <strong>em negrito</strong> assim.</p>
<p>E em <em>itálico</em> assim.</p>
CSS 
h1 {
  font-size: 48px;
  color: #1a1a2e;
  font-family: Arial, sans-serif;
}

p {
  font-size: 16px;
  color: #444444;
  line-height: 1.6; /* espaço entre linhas */
}

03

Cores e gradientes

Cores podem ser escritas pelo nome (red, blue), por código hexadecimal (#ff0000) ou por RGB. Gradientes misturam duas ou mais cores — ótimo para fundos!
CSS — Cor sólida 
body {
  background-color: #f0f4ff; /* fundo da página */
  color: #222222;           /* cor do texto */
}

h1 {
  color: #e63946;  /* vermelho vibrante */
}
CSS — Gradiente no fundo 
.caixa-colorida {
  background: linear-gradient(135deg, #667eea, #764ba2);
  /* 135deg = ângulo; depois as cores, separadas por vírgula */
  padding: 40px;
  color: white;
  border-radius: 12px;
}
HTML 
<div class="caixa-colorida">
  <h2>Texto com fundo gradiente 🎨</h2>
</div>
resultado no navegador

Texto com fundo gradiente 🎨

🎨 Use o site uigradients.com para descobrir combinações de cores bonitas e copiar o código CSS pronto!
04

Centralizar elementos

Centralizar é uma das coisas mais comuns num site. Veja as duas formas mais usadas:
CSS — Centralizar texto 
h1 {
  text-align: center; /* centraliza o texto */
}

p {
  text-align: center;
}
CSS — Centralizar um bloco na tela (Flexbox) 
.container {
  display: flex;
  justify-content: center;  /* centraliza na horizontal */
  align-items: center;      /* centraliza na vertical */
  height: 200px;
  background: #f0f4ff;
}

.caixa {
  background: #4f9cf9;
  color: white;
  padding: 20px 40px;
  border-radius: 10px;
}
HTML 
<div class="container">
  <div class="caixa">
    <p>Estou centralizado! 🎯</p>
  </div>
</div>
resultado no navegador
Estou centralizado! 🎯
05

Navbar (menu de navegação)

A navbar fica no topo do site e tem os links para as páginas. Ela usa a tag <nav> e links com a tag <a>.
HTML 
<nav class="navbar">
  <div class="logo">MeuSite</div>
  <ul class="menu">
    <li><a href="index.html">Início</a></li>
    <li><a href="sobre.html">Sobre</a></li>
    <li><a href="contato.html">Contato</a></li>
  </ul>
</nav>
CSS 
.navbar {
  background: #1a1a2e;
  display: flex;
  justify-content: space-between; /* logo à esq, menu à dir */
  align-items: center;
  padding: 0 40px;
  height: 60px;
}

.logo {
  color: white;
  font-size: 22px;
  font-weight: bold;
}

.menu {
  list-style: none; /* remove os pontinhos da lista */
  display: flex;
  gap: 30px;
}

.menu a {
  color: #aaaaaa;
  text-decoration: none; /* remove o sublinhado */
  font-size: 15px;
  transition: color 0.2s;
}

.menu a:hover { /* efeito ao passar o mouse */
  color: white;
}

06

Botões estilizados

A tag <button> cria um botão clicável. Com CSS, você pode deixar ele bonito e com efeito ao passar o mouse.
HTML 
<button class="btn-primario">Clique Aqui</button>
<button class="btn-secundario">Saiba Mais</button>
CSS 
.btn-primario {
  background: #4f9cf9;
  color: white;
  border: none;
  padding: 12px 28px;
  border-radius: 8px;
  font-size: 16px;
  cursor: pointer; /* muda o cursor para mãozinha */
  transition: background 0.2s;
}
.btn-primario:hover {
  background: #2563eb; /* azul mais escuro ao passar o mouse */
}

.btn-secundario {
  background: transparent;
  color: #4f9cf9;
  border: 2px solid #4f9cf9;
  padding: 12px 28px;
  border-radius: 8px;
  font-size: 16px;
  cursor: pointer;
}

07

Imagens

A tag <img> exibe imagens. O atributo src é o endereço da imagem. Use imagens da sua pasta ou da internet.
HTML 
<!-- Imagem da sua pasta (coloque junto com o index.html) -->
<img src="foto.jpg" alt="Minha foto" class="foto">

<!-- Imagem da internet (use a URL completa) -->
<img src="https://picsum.photos/400/300" alt="Foto aleatória" class="foto">
CSS 
.foto {
  width: 100%;          /* ocupa toda a largura disponível */
  max-width: 400px;    /* mas não passa de 400px */
  border-radius: 12px; /* cantos arredondados */
  display: block;
  margin: 0 auto;      /* centraliza a imagem */
}
💡 Sempre use o atributo alt com uma descrição da imagem. Isso ajuda pessoas com deficiência visual e é bom para o Google também!
08

Cards (cartões de conteúdo)

Cards são caixas que organizam informações de forma visual. São muito usados em sites de notícias, portfólios e lojas.
HTML 
<div class="card">
  <img src="https://picsum.photos/300/180" alt="Foto">
  <div class="card-conteudo">
    <h3>Título do Card</h3>
    <p>Uma descrição curta sobre o assunto do card.</p>
    <button>Ver mais</button>
  </div>
</div>
CSS 
.card {
  background: white;
  border-radius: 12px;
  box-shadow: 0 4px 20px rgba(0,0,0,0.1);
  overflow: hidden; /* a imagem não sai dos cantos */
  max-width: 300px;
  transition: transform 0.2s;
}
.card:hover {
  transform: translateY(-5px); /* sobe um pouco ao hover */
}

.card img {
  width: 100%;
  display: block;
}

.card-conteudo {
  padding: 20px;
}

09

Flexbox — Organizar itens lado a lado

O Flexbox é a ferramenta mais usada para organizar elementos em linha. Com ele você coloca cards lado a lado, cria menus, etc.
HTML 
<div class="linha-de-cards">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
</div>
CSS 
.linha-de-cards {
  display: flex;
  gap: 20px;             /* espaço entre os itens */
  flex-wrap: wrap;       /* quebra a linha em telas menores */
  justify-content: center;
}

.card {
  background: #4f9cf9;
  color: white;
  padding: 30px 40px;
  border-radius: 10px;
  font-size: 18px;
  font-weight: bold;
}

10

Mudar de página no site

Para ter mais de uma página, crie vários arquivos HTML na mesma pasta e use a tag <a> para criar links entre eles.
📁 Estrutura de pastas 
meu-site/
├── index.html      ← página principal
├── sobre.html      ← página "Sobre"
├── contato.html    ← página "Contato"
└── estilo.css      ← arquivo de estilos
HTML — Links entre páginas 
<!-- Em qualquer página, crie links assim: -->
<a href="index.html">Ir para Início</a>
<a href="sobre.html">Ir para Sobre</a>
<a href="contato.html">Ir para Contato</a>

<!-- Link para um site externo: -->
<a href="https://google.com" target="_blank">
  Abrir Google em nova aba
</a>
💡 target="_blank" faz o link abrir em uma nova aba. Use para links externos (outros sites). Para páginas do seu próprio site, não é necessário.
⚠️ Os arquivos precisam estar na mesma pasta! Se criar subpastas, o endereço muda: href="pages/sobre.html"
11

Fontes do Google Fonts

O Google oferece centenas de fontes gratuitas. Basta adicionar um link no <head> e usar no CSS!
HTML — Adicionar no <head> 
<head>
  <!-- Cole esse link antes do link do CSS -->
  <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap" rel="stylesheet">
  <link rel="stylesheet" href="estilo.css">
</head>
CSS — Usar a fonte 
body {
  font-family: 'Poppins', sans-serif;
}

/* O 'sans-serif' é o plano B caso a fonte não carregue */
🔤 Acesse fonts.google.com, escolha uma fonte, clique em "Get font" e depois copie o código que o site gerar para você.
12

Site responsivo (funcionar no celular)

Um site responsivo se adapta ao tamanho da tela. Usamos media queries no CSS para mudar o estilo em telas menores.
HTML — Tag obrigatória no <head> 
<!-- Sempre inclua essa linha no head! -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
CSS — Media Queries 
/* Estilo padrão (desktop) */
.linha-de-cards {
  display: flex;
  gap: 20px;
}

.navbar {
  padding: 0 40px;
}

/* Quando a tela for menor que 600px (celular) */
@media (max-width: 600px) {

  .linha-de-cards {
    flex-direction: column; /* empilha os cards */
  }

  .navbar {
    padding: 0 16px; /* menos espaço nas laterais */
  }

  h1 {
    font-size: 28px; /* título menor no celular */
  }
}
📱 No navegador, pressione F12 e clique no ícone de celular para simular como seu site aparece no smartphone!
13

Como conectar o arquivo CSS ao HTML

O CSS normalmente fica em um arquivo separado. Isso ajuda a organizar o projeto e reutilizar estilos em várias páginas.
📁 Estrutura de arquivos 
meu-site/
├── index.html
└── styles.css
HTML — Importando o CSS 
<head>

  <!-- Conecta o arquivo CSS ao HTML -->
  <link
    rel="stylesheet"
    href="styles.css"
  >

</head>
CSS — styles.css 
body {
  background: #0f172a;
  color: white;
  font-family: Arial, sans-serif;
}

h1 {
  color: #4f9cf9;
}
resultado no navegador

Meu Site

O CSS foi carregado com sucesso 🎉

💡 O atributo href informa o caminho do arquivo CSS. Se o CSS estiver em outra pasta, o caminho muda: href=\"css/styles.css\"
⚠️ O arquivo CSS precisa existir no local correto. Se o nome ou caminho estiver errado, o estilo não será aplicado.
